# Burger Dash Sprint

## Sprint Goal
Make the current Burger Dash run feel cleaner, fairer, and more legible while strengthening the burger-joint/chaser identity.

## Sprint Priorities
1. Gameplay fairness and readability at speed
2. Chaser pressure and catch presentation
3. HUD/title/game-over identity polish
4. Character select/progression groundwork only if the above are stable

## Sprint Definition Of Success
- The player can read turns, obstacles, and coins more clearly at speed.
- The chaser feels intentional and threatening, not placeholder.
- The game presents a coherent burger-joint fantasy.
- Every autonomous cycle leaves behind updated memory files.
