# Burger Dash Roadmap

## Phase 1: Core Run Must Be Fun
- Obstacle fairness
- Coin readability
- Turn feel
- Camera feel
- Speed tuning
- Hit / landing / checkpoint feedback
- Chaser pressure prototype

## Phase 2: Identity Must Be Obvious
- Burger-joint environment polish
- Stronger villain/chaser personality
- HUD/title/game-over polish
- Better visual quality for coins and obstacles

## Phase 3: Retention Layer
- Character select wiring
- Character progression / unlocks
- Selected character persistence
- Character preview and loading flow

## Phase 4: Expansion Systems
- Power-ups
- Combo / score systems
- Audio expansion
- Theme variety

## Phase 5: Platform Hardening
- Performance profiling
- Pooling / culling strategy
- PWA hardening
- Mobile/iPhone test pass
